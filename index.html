HTML as One of My Core Skills

HTML stands for HyperText Markup Language. It is used to create and structure web pages.

Structural Elements

The html tag is the root of a webpage. The head tag contains metadata, while the body tag contains visible content.

Headings

Heading tags (h1 to h6) are used to define titles. h1 is the most important heading.

Paragraph

The p tag is used to write paragraphs of text.

Lists

The ul tag creates unordered lists, while ol creates ordered lists. li is used for list items.

Links

The a tag is used to create hyperlinks. It uses the href attribute to define the destination.

Images

The img tag is used to display images. The src attribute specifies the image path, and alt describes the image.

Tables

The table tag is used to display data in rows and columns. tr defines rows, th defines headers, and td defines cells.

Forms

The form tag is used to collect user input. Input fields like text, email, and textarea allow users to enter data.

Semantic Elements

Semantic HTML elements clearly describe their meaning. Examples include header, footer, and section. They improve readability and structure.
